# E-commerce Data Analytics Project

Proyecto de análisis de datos de ventas de e-commerce utilizando:

- SQL (Data Warehouse)
- Python
- Pandas
- Google Colab

## Estructura

data/
Dataset de ventas.

notebooks/
Análisis exploratorio y visualización.

sql/
Scripts de creación del Data Warehouse.

## Dataset

El dataset contiene información de:

- ventas
- clientes
- productos
- categorías
- fechas
